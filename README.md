# Дипломная работа по профессии «Frontend-разработчик»

### Цели дипломной работы

В этой дипломной работе я создадала сайт для бронирования билетов в кинотеатр и разработала информационную систему для администрирования залов, сеансов и предварительного бронирования билетов.


### Функциональность

Клиентская часть (Гость)

- Просмотр расписания сеансов

- Просмотр информации о фильмах

- Выбор места в кинозале

- Бронирование билетов

- Генерация QR-кода с информацией о билете


### Административная часть

- Конфигурация залов (схема мест)

- Управление типами мест (обычные/VIP)

- Настройка цен на билеты

- Управление фильмами и сеансами


### Технологический стек

Frontend

HTML5 - семантическая верстка

CSS3 - кастомные стили, адаптивная верстка

JavaScript (ES6+) - клиентская логика

QRCreator.js - генерация QR-кодов


### Архитектура

Local Storage - хранение данных бронирования

URL Parameters - передача данных между страницами

Modular CSS - БЭМ-методология

### Страницы приложения

# Клиентские страницы

- index.html - Главная страница с расписанием
  
- booking.html - Выбор мест в зале
  
- payment.html - Подтверждение бронирования
  
- ticket.html - Получение билета
  

# Админские страницы  

- login.html - Страница входа
  
- index.html - Панель управления



Административные функции

- Конфигурация схемы залов

- Управление типами мест (free/vip/blocked)

- Настройка цен

- Управление сеансами


### Ключевые особенности

Бронирование билетов

Интерактивная схема зала с визуализацией мест

Разные типы мест: обычные (250 руб) и VIP (350 руб)

Валидация выбора мест

Расчет общей стоимости

QR-система

Генерация уникальных QR-кодов для каждого билета

Информация в QR-коде: фильм, места, зал, время, стоимость

Резервный вариант при ошибке генерации


### Адаптивность

Responsive дизайн (от 320px)

Кроссбраузерная совместимость

Оптимизация для мобильных устройств
